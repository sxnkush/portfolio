ConvCurrency

Built using React and styled with Tailwind CSS, this currency converter application allows users to effortlessly convert between various currencies. By fetching live exchange rates from an external API, the app provides up-to-date data for each currency. Users simply input their desired amount and select their currencies, and the converter calculates the total in real-time, ensuring accuracy and a smooth user experience. This project demonstrates dynamic API integration, responsive UI design, and user-focused functionality for practical, everyday currency conversions.

WorkOnIt

Users can add new tasks, edit existing ones, and delete tasks as needed. Tic Tac Toe

The Tic Tac Toe Game is a web-based implementation of the classic two-player game, designed to be played in a browser environment. Comprising HTML, CSS, and JavaScript, this project encapsulates the essence of simplicity and interactivity. Upon loading, users are presented with a 3×3 grid where they take turns marking spaces with either ‘X’ or ‘O’. The game dynamically updates the board, validates results, and determines the winner based on horizontal, vertical, or diagonal alignments.
